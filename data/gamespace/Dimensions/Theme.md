# {{Theme}}

## Definition
What aspect of the game better expresses the game's core intent and vibe.
Scalar on:
How important is for the game experience that you connect with the given aspect?


## Additional Notes
Trying to approximate to the aspect of the game that informs the theme rather than the theme itself in order to allow for the exploration of similar pieces expressing different themes. Purposefully conflates hard mechanics and dynamics (Skill Tree, Dialogue) with general vibes and aesthetics (Organization, Philosophy). A theme should contribute to the development of the game's core appeal so that the "overall" theme can be inferred from the sum of the listed themes and all of the list can be assumed to be relevant. (Trying to avoid listing themes that are only marginally relevant or don't add much to the experience, that means the attribute cannot be scaled on the basis on how relevant the theme is, which was a flawed idea anyways)
