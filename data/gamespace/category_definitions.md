# Category Definitions

## Tag Taxonomy

### Domain Tags
Categorize what aspect of the game experience the parameter measures:

- `#gameplay` - Core mechanical/interactive elements
- `#narrative` - Story, worldbuilding, character, themes
- `#progression` - How the game unfolds over time, pacing
- `#meta` - Structural properties (replayability, openness, loop design)
- `#emotional` - Tone, atmosphere, emotional impact
- `#challenge` - Difficulty, hostility, player skill requirements

### Granularity Tags
How specific or composite is this parameter?

- `#atomic` - Measures one clear, specific thing
- `#composite` - Might actually be measuring multiple concepts
- `#fuzzy` - Unclear boundaries or definition

### Overlap Risk Tags
Relationship to other parameters:

- `#high-overlap` - Significant overlap with other parameters
- `#distinct` - Clearly measures something unique
- `#redundant` - Might be fully captured by other parameters

### Priority Tags
How important is this to the recommendation system?

- `#core` - Essential for good recommendations
- `#secondary` - Useful but not critical
- `#questionable` - Unclear value add

### Decision Stage Tags
Where you are in evaluation:

- `#evaluating` - Currently working on this
- `#keep` - Decided to keep
- `#modify` - Needs refinement but keep
- `#collapse` - Combine with another parameter
- `#remove` - Cut from the system
